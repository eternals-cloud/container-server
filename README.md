# Logistics Server

### Version

> 1.0

## Description

A logistics management system (LMS) is a combination of software tools that optimize business processes. It can help with: Order processing, Inventory management, Transportation, Returning orders, Warehousing.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Environment Variable

```
PORT=5100
```
